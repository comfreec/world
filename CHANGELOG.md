# 변경 이력 (Changelog)

## [1.0.0] - 2024-02-27

### 🎉 초기 릴리즈

#### 주요 기능
- 🎯 국기 퀴즈 게임
  - 3가지 게임 모드: 일반, 타임어택(60초), 서바이벌(3번 실수)
  - 4가지 난이도: 쉬움(10), 보통(20), 어려움(30), 도전(50)
  - 선택형 및 입력형 모드 지원
  - 음성 안내 및 효과음

- 🧠 메모리 게임
  - 2인 대전 모드
  - 16쌍의 국기 카드 매칭
  - 최고 기록 저장

- 📚 학습 모드
  - 195개 국가의 국기 탐색
  - 즐겨찾기 기능
  - 검색 기능 (초성 검색 지원)
  - 대륙별 필터링

- 🏆 성취 시스템
  - 27개의 다양한 업적
  - 정답 수, 연속 정답, 게임 모드별 업적
  - 시간대별 플레이 업적 (아침, 밤)
  - 연속 플레이 일수 추적

- 📊 통계 시스템
  - 게임별 최고 점수 추적
  - 총 정답 수 및 게임 수
  - 학습한 국가 수
  - 연속 정답 기록
  - 시간대별 플레이 통계

#### 설정 및 기능
- ⚙️ 설정 페이지
  - 음성 안내 on/off
  - 효과음 on/off
  - 음성 속도 조절 (기본 1.3배속)

- 📱 PWA 지원
  - 오프라인 모드
  - 홈 화면에 추가
  - 빠른 로딩

- 📄 법적 문서
  - 개인정보처리방침
  - 이용약관
  - 도움말 페이지
  - 앱 정보 페이지

#### 기술적 개선
- Service Worker 구현 (오프라인 지원)
- LocalStorage 기반 데이터 저장
- Web Speech API (음성 안내)
- Web Audio API (효과음)
- 반응형 디자인 (모바일 최적화)

#### UI/UX
- 깔끔한 그라디언트 디자인
- 직관적인 네비게이션
- 애니메이션 효과
- 접근성 개선 (ARIA 레이블)
- SEO 최적화 (메타 태그)
