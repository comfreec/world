### 2. TWA (Trusted Web Activity) 설정
Google Play Store에 PWA를 배포하려면 TWA를 사용해야 합니다.

#### 필요한 도구
- Android Studio
- Bubblewrap CLI (권장)

#### Bubblewrap 사용 방법
```bash
# Bubblewrap 설치
npm install -g @bubblewrap/cli

# TWA 프로젝트 초기화
bubblewrap init --manifest https://comfreec.github.io/world/manifest.json

# APK 빌드
bubblewrap build

# Play Store용 AAB 빌드
bubblewrap build --buildType=release
```
