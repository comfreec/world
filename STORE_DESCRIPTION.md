  - 2인 대전 모드

  - 최고 기록 저장

• 학습 모드
  - 195개 국가 탐색
  - 즐겨찾기 기능
  - 대륙별 필터링
  - 검색 기능

• 성취 시스템
  - 27개의 다양한 업적
  - 정답 수, 연속 정답 업적
  - 게임 모드별 업적
  - 시간대별 플레이 업적

• 통계 추적
  - 게임별 최고 점수
  - 학습한 국가 수
  - 연속 정답 기록
  - 시간대별 통계

## 🇺🇸 English Version

### App Name
World Flag Master

### Short Description (80 chars)
Learn flags of 195 countries with fun educational games

### Long Description
World Flag Master is an educational game app where you can learn flags of 195 countries around the world in a fun way.

🎯 Main Features:

• Flag Quiz
  - Normal Mode: Choose 10-50 questions
  - Time Attack: Score as much as you can in 60 seconds!
  - Survival: Challenge with 3 lives!
  - Multiple choice & Input modes

• Memory Game
  - 2-player mode
  - Match 16 pairs of flag cards
  - High score tracking

• Learning Mode
  - Explore 195 countries
  - Bookmark favorites
  - Filter by continent
  - Search function

• Achievement System
  - 27 diverse achievements
  - Correct answers & streak achievements
  - Game mode achievements
  - Time-based play achievements

• Statistics Tracking
  - High scores by game
  - Countries learned
  - Max streak records
  - Time-based statistics

✨ Features:
- 🌐 Offline support
- 🔊 Voice guidance & sound effects
- 🎨 Clean design
- 💯 Free, no ads
- 📱 Fast loading with PWA technology

👨‍👩‍👧‍👦 Perfect for:
- Students learning geography
- Travel enthusiasts
- Brain training seekers
- Parents educating children

Start your world tour with World Flag Master now! 🌍
